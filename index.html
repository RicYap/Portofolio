Supermarket Branch Analysis



Year
2022

Role
- Data Analyst

Repository Link
https://github.com/RicYap/Supermarket-Branch-Analysis

Kaggle
https://www.kaggle.com/code/rickyaryatama/supermarket-branch-analysis

Supermarket Branch Analysis is a project from Big Data Processing course. The objective of the project is to analyst data and get useful information from the data for the supermarket store. In this case my group using cluster analysis to process data in order to see the grouping of the data. First step is plotting the data using scatter plot to see data distribution. Second step is using min-max scaler to make the data more scalable. Third step is using elbow method to find the best quantity of group for clustering. The last step is to make the group using K-means clustering method.The code is using python language and coded in kaggle platform.
